Descripción del proyecto

Esta API es la encargada de gestionar los usuarios, recolectar datos sobre el gameplay y la leaderboard. Toda la información es almacenada en una base de datos MySQL en el servidor.

Actualmente se encuentra en producción y es usada por la aplicación móvil de Leba, que cuenta con mas de 2000 descargas en la Play Store.
